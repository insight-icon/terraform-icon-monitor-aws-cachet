## Features

This module sets up a cachet status page on AWS / docker with Ansible.  SSL cert with letsencyrpt.  

## Terraform Versions

For Terraform v0.12.0+

## Usage

```hcl-terraform
module "this" {
source = "github.com/insight-icon/terraform-icon-monitor-aws-cachet"
}
```
